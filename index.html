About this repository.

This repository is compatible with all package managers including Cydia, Sileo, Installer and Zebra. It also has beautiful depictions on modern package managers.

It's recommended to use Sileo while browsing through packages because this repo is meant for modern package managers.

How to add my repo
Easy steps on how to add my repo

Address to add: https://dylbinthedev.github.io

  • Cydia

    Launch Cydia and tap the Sources tab, then tap the + and enter https://dylbinthedev.github.io

  • Sileo

    Launch Sileo and tap the Sources tab, then tap the + and enter https://dylbinthedev.github.io

  • Installer

    Launch Installer and tap the Sources tab, then tap the + and enter https://dylbinthedev.github.io

  • Zebra

    Launch Zebra and tap the Sources tab, then tap the + and enter https://dylbinthedev.github.io
